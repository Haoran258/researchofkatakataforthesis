## 📖 项目简介 / Introduction

本项目是面向**印度尼西亚汉语学习者**（HSK 4-5 级）的中介语偏误语料库，共收录 **2000 条** 真实及高仿真偏误条目，覆盖语序、用词、搭配、句法残缺、冗余结构与母语词汇替代六大主类，按印尼坤甸地区某语言学院 2025-2026 学年的真实教学场景（期中/期末考试、课堂写作、学生日记、单元测试等）进行多维度元数据标注。

This corpus contains 2,000 interlanguage error entries from Indonesian HSK 4-5 learners, annotated with rich metadata including error type, source, time, classroom, and Indonesian-Chinese contrastive interference notes. Suitable for SLA research, CSL teaching, error analysis, and NLP/grammar-checking model training.
